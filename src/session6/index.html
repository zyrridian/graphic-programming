Session 6

Camera Lab

This session demonstrates how one scene can feel very different depending on the active camera control strategy.

  • 1 = manual keyboard movement
  • 2 = OrbitControls for object inspection
  • 3 = FirstPersonControls for immersive walking
  • 4 = FlyControls for free 3D navigation
The skybox and grid stay constant so the effect of each camera mode is easier to compare.
